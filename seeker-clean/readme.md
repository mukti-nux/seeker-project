# Seeker Vercel Edition

Tool pelacakan lokasi berbasis browser. Dihosting dengan Vercel, mengarahkan user ke link setelah mengirim lokasi.

## Fitur
- Deteksi lokasi via browser
- Kirim data ke server
- Redirect otomatis
- Siap deploy ke Vercel

## Deploy
1. Fork / Clone
2. Deploy ke [Vercel](https://vercel.com/import)
